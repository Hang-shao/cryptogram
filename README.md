# SM2
（1）sm2_点集
求椭圆曲线上的点，例：y^2=x^3+ax+b，求(x,y)

（2）sm2_逆运算
求一个数的逆，例：求Fq下的g^-1

（3）sm2_平方根
求椭圆方程中的y值，即求平方根，例：y^2=g(mod p)，求y

（4）sm2_指数运算
求一个数的指数值，例：求Fq下的g^a
